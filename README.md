## 📦 Instalación / Uso local

Si deseas ejecutar el proyecto en tu computadora:

### 1. Clonar el repositorio

```bash
git clone https://github.com/lacreamno1/Proyecto-Becalos-2.git
```

### 2. Entrar al proyecto

```bash
cd Proyecto-Becalos-2
```

### 3. Abrir el proyecto

* Abre el archivo `index.html` en tu navegador
* O utiliza una extensión como **Live Server** en VS Code
