Acerca del proyecto

¿Qué es este proyecto?

Este sitio web forma parte de mi proyecto final para el curso Bécalos – Introducción a la Programación. El objetivo es aplicar los conocimientos adquiridos en HTML, CSS y conceptos básicos de desarrollo web para construir una página con estructura clara, diseño atractivo y contenido organizado.

Objetivos

  • Practicar la creación de páginas web con HTML semántico.
  • Aplicar estilos con CSS para lograr un diseño elegante.
  • Reforzar buenas prácticas de accesibilidad y organización.
  • Mostrar creatividad con una paleta morado y dorado.

Aprendizajes

A través de este proyecto estoy reforzando habilidades como:

  • Organización de contenido en secciones y tarjetas.
  • Uso de tipografía y colores para transmitir identidad.
  • Construcción de menús de navegación entre páginas.
  • Preparación para proyectos más complejos en el futuro.
